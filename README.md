# 正史編纂室

ChatGPTの`conversations.json`をブラウザ内だけで読み込む、静的なローカルファーストPWAの試作版です。

## v2の機能

- `current_node`から親をたどり、選択中だった会話枝を表示
- 全文検索と最初の一致箇所への移動
- 応答メタデータに残るモデルで絞り込み
- タイトル・本文から仮推定したペルソナで絞り込み
- オフライン起動
- 複数JSONの差分統合と端末内保存
- 話者名表示・表示名設定・匿名化置換
- セッションのフォルダ分類とペルソナ修正
- 発言単位の編集・非表示・復元
- セッション単位のMarkdown／JSON出力

ログはアップロードせず、ブラウザの端末内ストレージだけで処理します。
